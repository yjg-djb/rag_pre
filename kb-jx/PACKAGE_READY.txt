========================================
内网部署步骤
========================================

1 传输文件
   - 通过 U盘/内网 FTP 传输 ZIP 包
   - 解压到目标目录

2 安装依赖
   - 双击运行: install_offline.bat
   - 或手动安装: pip install --no-index --find-links=offline_packages -r requirements.txt

3 验证安装
   - 双击运行: verify_install.bat
   - 检查所有依赖是否安装成功

4 启动服务
   - 双击运行: start.bat
   - 或执行: python main.py

5 访问系统
   - 浏览器打开: http://localhost:8000
   - 上传页面: http://localhost:8000/upload
   - API 文档: http://localhost:8000/docs
